CSS Diner

Lista de selectores

      • A: Selecciona todos los elementos del tipo especificado como A.
      • #id: Selecciona todos los elementos que tienen el identificador especificado.
      • A B: Selecciona todos los elementos de tipo B que son descendientes directos de un elemento A.
      • .classname: Selecciona todos los elementos que tienen la clase especificada.
      • A, B: Combinador de selectores que selecciona todos los elementos que coinciden con A y B.
      • *: Selector universal que selecciona todos los elementos en el documento.
      • A + B: Selecciona todos los elementos de tipo B que son hermanos directos de un elemento A.
      • A ~ B: Selecciona todos los elementos de tipo B que son hermanos de un elemento A, seguidos inmediatamente por él.
      • A > B: Selecciona todos los elementos de tipo B que son hijos directos de un elemento A.
      • :first-child: Selecciona el primer hijo de un elemento padre.
      • :only-child: Selecciona los elementos que son los únicos hijos de sus padres.
      • :last-child: Selecciona el último hijo de un elemento padre.
      • :nth-child(A): Selecciona el Aº hijo de un elemento padre.
      • :nth-last-child(A): Selecciona el Aº hijo de un elemento padre, contando desde el último hijo.
      • :first-of-type: Selecciona el primer elemento de su tipo dentro de un elemento padre.
      • :nth-of-type(A): Selecciona cada Aº elemento de su tipo dentro de un elemento padre, incluyendo elementos pares e impares.
      • :nth-of-type(An+B): Selecciona todos los elementos de su tipo a partir del elemento número B, con incrementos de A.
      • :only-of-type: Selecciona el único elemento de su tipo dentro de un elemento padre.
      • :last-of-type: Selecciona el último elemento de su tipo dentro de un elemento padre.
      • :empty: Selecciona elementos que no tienen ningún contenido o descendientes directos.
      • :not(X): Selecciona todos los elementos que no coinciden con el selector especificado como X.
      • [attribute]: Selecciona todos los elementos que tienen el atributo especificado.
      • A[attribute]: Selecciona todos los elementos A que tienen el atributo especificado.
      • [attribute="value"]: Selecciona todos los elementos que tienen el atributo con el valor exacto especificado.
      • [attribute^="value"]: Selecciona todos los elementos que tienen el atributo cuyo valor comienza con la cadena especificada.
      • [attribute$="value"]: Selecciona todos los elementos que tienen el atributo cuyo valor termina con la cadena especificada.
      • [attribute*="value"]: Selecciona todos los elementos que tienen el atributo cuyo valor contiene la cadena especificada.
